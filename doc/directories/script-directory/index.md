# Script directory

To get the directory of a running script:

```sh
dir() { CDPATH= cd -- "$(dirname -- "$0")" && pwd -P ; }
```

This solution is POSIX. However, it's not perfect; it has issues with symlinks, and also does not work if the script is sourced.
